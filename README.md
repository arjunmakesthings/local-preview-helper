small script to locally preview on mac / any other device by writing a single word on the terminal.

---
### install: 
include the script with: 

`nano ~/.zshrc`

this opens an editor. copy paste the script inside it. 

press `ctrl + o` to write out -> `enter` to write on the same file -> `ctrl + x` to exit. 

---
### usage:
now on any directory, write `serveme` (or whatever you named it), and it'll start a server and return your ip so that you can preview the same thing on your phone with `ip:port`. port is set as `8000` by default (unless you change it in the script).
